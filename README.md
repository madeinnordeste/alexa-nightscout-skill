# alexa-nightscout-skill

Alexa skill para leiutura de dados do Nightscout

## NowIntent
Utterances:
* agora
* qual o valor
* quanto está
* presente
* sobre o presente
* este momento
* neste momento

## NextIntent
Utterances:
* futuro
* sobre o futuro
* próximos minutos
* breve
* em breve

## PrevIntent
Utterances:
* passado
* do passado
* anteriormente
* últimas horas
